# 外部参考与搬运清单

## 延迟模拟器 TSC 忙等

- 来源：`../cxlkv/src/utils/src/latency_simulator.cc`，参考版本见 `PLAN.md` M0
  记录的 cxlkv SHA。
- 落点：`kv/engine/latency_inject.cpp` 与 `kv/engine/latency_inject.h`。
- 改写：逐文件移植 `latency_sim` 的 TSC 校准、线程本地 generation、组相联 LRU、
  scope 与统计 API；只调整本仓 include 路径，不引入对外部工程的构建或运行时依赖。

## VM 编排脚本

- 来源：参考 `../cxlkv` 的 VM 初始化职责与 QEMU/ivshmem 参数合同，具体版本见
  `PLAN.md` M0 记录。
- 落点：`tigonkv_make_vm_img.sh`、`tigonkv_init_vms.sh`、
  `tigonkv_kill_vms.sh`、`tigonkv_check_vms.sh` 与 `scripts/tigonkv_vm_common.sh`。
- 改写：使用本仓库的 mkosi、配置和 pid 文件；默认状态变更受
  `--allow-state-change` 闸门保护，未引入对兄弟工程脚本、镜像或运行时文件的依赖。

## YCSB 编排与汇总

- 来源：参考 `../cxlkv/scripts/run_ycsb_trace_experiment.sh` 的参数与产物合同，
  具体版本见 `PLAN.md` M0 记录。
- 落点：`tigonkv_run_ycsb_experiment.sh`、`scripts/summarize_ycsb_experiment.py`
  和 `YCSB指南.md`。
- 改写：trace 仅经本仓 `thirdparty_libs/YCSB-cpp` 生成；VM 回放仅调用本仓
  `scripts/e2e_trace/run_guest_ycsb_workflows.sh`，并将未验收的 workload E 明确
  标为 unsupported。
